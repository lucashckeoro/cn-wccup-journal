# cn-wccup-journal

本仓库用于归档和发布多个独立 HTML 页面。页面内容来自不同来源，以静态文件形式存放，不针对任何特定域名或网站。

## 项目简介

这是一个以 HTML 页面归档为主要目的的仓库。仓库中的页面均为独立文件，可被直接查看或部署。本仓库不提供任何形式的网站聚合、导航或推荐服务，仅作为页面存档使用。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库入口页面（可选）
├── pages/             # 主要页面存放目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/            # 共用资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立 HTML 页面文件，每个文件为一个完整页面。
- `assets/`：存放页面可能引用的样式表、JavaScript 脚本、图片等资源文件。

## 页面归档说明

- 所有页面以 HTML 静态文件形式提交到仓库中。
- 页面之间无关联，各自独立运行。
- 页面内容可能包含第三方资源链接（如外部样式、字体、脚本等），但本仓库不对这些外部资源的可用性及内容负责。
- 归档的页面可能随时间更新或移除，恕不另行通知。

## 维护说明

- 欢迎通过 Issue 提交问题或建议，但不保证及时处理。
- 不接受未经沟通的直接 Pull Request，如需贡献请先讨论。

## 使用方式

你可以通过以下方式使用本仓库中的页面：

1. 直接克隆仓库到本地，在浏览器中打开 `pages/` 下的任意 HTML 文件。
2. 将仓库部署到任何静态托管服务（如 GitHub Pages、Netlify 等），即可在线访问。
